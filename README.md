# Library Information System
This project was my **Class 12 Capstone Project** at National High School (2023-24).

### Key Features
* **Automated Fine Calculation:** Implemented logic for tiered late fees based on return dates.
* **Data Visualization:** Uses Matplotlib to generate inventory graphs.
* **Database Management:** Originally built with MySQL, now modernized to use SQLite.

### Why this matters
This project demonstrates my early foundations in **Logic Design** and **CRUD operations**, which I am now expanding through my studies in **AI & Machine Learning** at HITK.
